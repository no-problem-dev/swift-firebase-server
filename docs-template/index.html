swift-firebase-server

Server-side Swift libraries for Firebase services.

  • FirestoreServer
    Firestore REST API client
  • FirestoreSchema
    Type-safe Firestore schema DSL with Swift macros
  • FirebaseStorageServer
    Cloud Storage for Firebase client
  • FirebaseStorageSchema
    Type-safe Storage schema DSL with Swift macros
  • FirebaseAuthServer
    Firebase ID token verification
